# PagerDuty Cursor Plugin

A Cursor plugin that connects your AI assistant to PagerDuty's remote MCP server, enabling incident management, on-call lookups, service management, and more.

## Overview

This plugin connects Cursor to PagerDuty's remote MCP server at `mcp.pagerduty.com/mcp`, enabling incident management, on-call lookups, service management, and more directly from your IDE.

## Installation

1. Open Cursor Settings
2. Navigate to **Plugins**
3. Click **Browse Marketplace**
4. Search for **PagerDuty**
5. Click **Install**
